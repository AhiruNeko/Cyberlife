# 🤖 Cyberlife - 深度强化学习数字生命体

基于深度Q学习(Deep Q Learning)的人工智能数字生命体，能在虚拟环境中自主学习和决策。

## 📋 项目概述

Cyberlife是一个创新的强化学习项目，创造了一个具有智能决策能力的虚拟生命体。通过Deep Q Learning算法的训练，数字生命体能够在复杂的环境中学习最优行为策略，展现出类似真实生物的适应性和学习能力。

## 🔧 技术栈

- **编程语言**: Python 100%
- **深度学习框架**: TensorFlow / Keras
- **算法**: 深度Q学习 (DQN)
- **环境模拟**: 自定义虚拟环境

## 🧠 核心算法

### 深度Q学习 (Deep Q Learning, DQL)

深度Q学习的核心方程式为：

$$Q(s,a)=r+\gamma \max Q_{target}(s', a';\theta^-)$$

其中：
- **s**: 当前状态（环境观察）
- **a**: 执行的动作
- **r**: 获得的奖励
- **γ (gamma)**: 折扣因子，用于平衡即时奖励和长期奖励
- **θ⁻**: 目标网络的权重参数

### ε-贪心 (ε-Greedy) 探索策略

在训练过程中，采用ε-贪心策略平衡探索(Exploration)与利用(Exploitation)：

$$action=\begin{cases}
\text{随机选择} & \text{概率} = \epsilon\\
\arg\max Q(s, a') & \text{概率} = 1-\epsilon
\end{cases}$$

其中ε会随时间衰减：

$$\epsilon=\max(\epsilon_{min}, \epsilon_{initial}\cdot k^t)$$

**参数含义**：
- **ε_initial = 0.99**: 初始探索概率（99%）
- **ε_min = 0.05**: 最小探索概率（5%）
- **k = 0.998**: 衰减系数
- **更新频率**: 每4步更新一次ε

## 🏗️ 神经网络架构

该项目采用的神经网络具有以下结构：

### 输入层
- **尺寸**: (3, 16, 1)
- **说明**: 包含当前时刻的环境观察，以及之前两个时刻的环境状态
- **目的**: 捕捉时间序列信息，帮助网络理解环境动态变化

### 特征提取部分

#### 卷积层 (Convolution2D)
- **过滤器数**: 32
- **核大小**: 1×1
- **激活函数**: ReLU (max(x, 0))
- **填充**: Same

#### 展平层 (Flatten)
- **功能**: 将2D卷积输出展平为1D向量

### 深度全连接部分

#### 全连接层 1 (Dense)
- **单元数**: 64
- **激活函数**: ReLU

#### 全连接层 2 (Dense)
- **单元数**: 128
- **激活函数**: ReLU

#### 丢弃层 (Dropout)
- **丢弃率**: 0.5
- **目的**: 防止过拟合

#### 全连接层 3 (Dense)
- **单元数**: 512
- **激活函数**: Sigmoid

#### 丢弃层 (Dropout)
- **丢弃率**: 0.5

#### 全连接层 4 (Dense)
- **单元数**: 64
- **激活函数**: ReLU

#### 输出层 (Dense)
- **单元数**: 6
- **激活函数**: Sigmoid
- **说明**: 输出6个Q值，对应6种可能的行为

## ⚙️ 训练参数详解

| 参数 | 值 | 说明 |
|------|-----|------|
| **ε_initial** | 0.99 | 初始探索概率 |
| **ε_min** | 0.05 | 最小探索概率 |
| **k** | 0.998 | 探索概率衰减系数 |
| **ε更新频率** | 4 | 每4步更新一次探索概率 |
| **γ (折扣因子)** | 0.3 | 平衡即时和长期奖励 |
| **主网络训练频率** | 4 | 每4步训练一次主网络 |
| **目标网络更新频率** | 40 | 每40步更新一次目标网络 |
| **经验回放缓冲区大小** | 5000 | 存储最近5000条经验 |

## 🎯 系统设计特点

### 1. **双网络架构**
- **主网络**: 用于学习Q值函数
- **目标网络**: 用于计算目标Q值，提高训练稳定性

### 2. **经验回放 (Experience Replay)**
- 随机采样进行训练，打破数据相关性
- 提高样本利用效率

### 3. **ε-衰减探索**
- 初期高度探索(99%随机)
- 逐步降低探索概率至5%
- 平衡学习过程中的探索和利用

### 4. **时间序列输入**
- 使用三帧的环境观察
- 捕捉环境动态和时间依赖性
- 增强网络对环境变化的理解

数字生命体可以执行6种行为：
1. 进食
2. 向前移动
3. 向后移动
4. 向左转
5. 向右转
6. 攻击
